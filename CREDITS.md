## Modifications

The original codebases have been substantially modified and unified to build Alfred:

- New unified user interface combining features from all three projects
- Mobile-responsive layout with Editor / Preview tab navigation
- Multilingual UI (English / Italian) with runtime language switching
- Ad Localizer tool (derived from Global Kit Generator)
- Magic Pixels AI photo editor (derived from Pixshop)
- Proactive prompt refinement with Belief Graph (derived from Proactive Co-Creator)
- Multi-mode generation (Image, Story, Video, Prompt) in a single application
- File and image attachment support
- Dark / Light mode toggle
